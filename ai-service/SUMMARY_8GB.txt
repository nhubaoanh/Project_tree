🚀 QUICK START (3 BƯỚC):

1️⃣  TĂNG VIRTUAL MEMORY (BẮT BUỘC!)
   
   Windows + R → sysdm.cpl → Enter
   Advanced → Settings → Advanced → Change
   
   Custom size:
   ┌─────────────────────────────┐
   │ Initial size: 12288         │
   │ Maximum size: 16384         │
   └─────────────────────────────┘
   
   Set → OK → RESTART MÁY

2️⃣  CÀI DEPENDENCIES
   
   cd ai-service
   pip install -r requirements.txt

3️⃣  CHẠY SERVICE
   
   python main.py
   
   Lần đầu: 5-10 phút (download model)
   Lần sau: 30-60 giây (load cache)
